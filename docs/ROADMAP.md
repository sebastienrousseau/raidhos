# Roadmap

## MVP

- Rust core disk discovery + safety checks.
- Cross-platform disk operations via helper.
- Install Ventoy payload to USB.
- Basic Tauri UI with disk selection and install progress.

## Beta

- ISO management with drag-and-drop.
- Integrity checks (optional SHA256).
- Better device heuristics and warnings.

## V1

- Persistence image creation.
- UI polish and theming.
- Packaging: Windows, macOS, Linux.
