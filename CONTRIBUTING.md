# Contributing to RaidhOS

Thanks for your interest in contributing! This project is early-stage and we welcome issues, ideas, and pull requests.

## Development setup

1. Install Rust (stable) and Cargo.
2. Clone the repo and enter it.
3. Build the workspace:

```bash
cargo build
```

## Code style

- Run `cargo fmt --all` before submitting.
- Keep changes focused and documented.

## Issues and PRs

- Please open an issue first for large changes.
- Include a clear description and the test steps you used.

## Security

If you find a security issue, please open a private report instead of a public issue.
